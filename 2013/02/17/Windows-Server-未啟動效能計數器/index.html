發表於 2013-02-17   |  

這幾天持續進行著IT的奇幻之旅，而這篇，也是隨手的一個小筆記 (  IT的前輩們不要笑QQ，身為Developer的小小弟我，其實是很難搞懂Server的複雜的XDD )

那怎樣解決呢!?其實很簡單，只要在伺服器管理員，選擇所有伺服器，往下拉，拉到下面這個畫面，並起選擇啟動效能計數器就可以了!!
