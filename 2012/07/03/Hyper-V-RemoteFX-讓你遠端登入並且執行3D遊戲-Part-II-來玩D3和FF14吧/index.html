Hyper-V - RemoteFX 讓你遠端登入並且執行3D遊戲! Part II 來玩D3和FF14吧!

03 July 2012 — Written by Sky Chang
#Hyper-V#Windows#Windows Server

之前有寫過一篇Hyper-V - RemoteFX 讓你遠端登入並且執行3D遊戲!，而因為當時家裡無線網路速度不快，再加上當時又沒有甚麼比較好玩遊戲可以測試，所以也只有跑跑3D Mark而已，而最近，終於有一個大作，也就是D3 ( diablo 3 暗黑破壞神3 )，所以當然就決定裝上去跑跑看嚕。

基本上，設定和前篇一樣，依據前篇的設定，原則上就可以跑起來，當然，CPU的部分要支援虛擬化，而如果想要在VM上的Win 7有Aero的效果，則必須在VM Server上裝上桌面體驗這個功能。

小弟的電腦配備為i7 920，24GRam，Nvidia GTX260的顯示卡，完全沒有跑Raid，而配置出來的VM是給四顆核心，4G Ram，虛擬的顯卡，並沒有給獨立的硬碟；而因為是用無線網路，所以連線過去，大概只有8MB/s左右 ( 大小的B喔 )。

執行出來的結果就如下圖，小弟是用Win 8遠端桌面登入到VM上的Win 7，並把Win 7上的Diablo 3跑起來，小弟沒跑Full HD就是了，單純的1280的解系度，而且特效的部分就是預設，所以下圖可以看到，沒有影子XDD，基本上，音效和移動都沒有甚麼大問題；但有時會小頓一下，畢竟還是沒有原本的順暢，而深度的職業玩家朋友和小弟說，你這才第一章，到後面你就會Lag到死了XDDD，但無奈，小弟等級不夠也沒辦法測試嚕。

接下來，是跑FF14 ( Final Fantasy XIV )的Benchmark，幾本上，FF14 Benchmark是需要跑DirectX 9，而預設的Win7是沒有裝DirectX Runtime，所以要先去這裡下載安裝DirectX End-User Runtime。

之後就可正常的跑測試，小弟第一次是只開兩顆核心，其餘配置都一樣，結果跑出來的分數為1667，第二次打開四核心，結果跑出來的分數如下圖，1700 ( 剛剛好XDD )，所以基本上CPU的多寡果然不是最重要的…，我們也可以看到下圖，顯示卡為Microsoft RemoteFX Graphics。

其實之前也還裝過三國志12日文測試版本，但後來沒留作紀錄就砍掉了QQ，那時有用3G網路連回去，老實說，真的很慢…。

後記

這篇文章，未來有新的東西，應該還會持續在這篇上面更新，這陣子還會真的把FF14裝上去跑跑看，並且改用有線網路測試看看，但不管如何，Windows Server 的Remote Service也越來越強大，隨著網路越來越快，我們上班偷玩3D遊戲，也越來越不是夢了XDD。
