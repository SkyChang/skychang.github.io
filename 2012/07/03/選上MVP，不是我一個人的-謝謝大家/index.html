選上MVP，不是我一個人的!謝謝大家!

03 July 2012 — Written by Sky Chang
#Sky的IT碎碎念

2012/7/1晚上，算是一個人生相當重要的里程碑，因為在那個時刻，我收到了微軟的MVP當選通知，老實說，從來沒有想過人生會當選微軟的MVP，從以前懵懵懂懂的學生時代，開始寫著ASP，途中遇到一堆問題時，從搜尋引擎查到MSDN，看著各個前輩的回答，重點是，那些人都掛著MVP的頭銜( 好啦，我承認印象最深的是小朱前輩XDD )；買著書的時候，看著MVP的Logo，到現在，那些書依然塵封在書櫃之中，而那些記憶，也留存在那少年時代的腦海裏面。

MVP就如前面所說的，一直到現在，還是覺得那是遙不可及的，老實說，今天能當選，一切也都是偶然中的偶然XDD，一切覺得是那樣的不可思議；而這一切，第一個要感謝的其實當然是家裡的老婆大人，和兒子臭笨小乖( 因為他都會罵我臭笨爸比= = )，每當深夜大家累了，想睡覺的時候，還能忍受我鍵盤的KILIKILI，寫著文章的聲音；而每次我在忙的時候，臭笨小乖還能乖乖地拿著iPad自己在那邊玩 ( 其實他比較愛iPad… )，也因為有家人的支持，所以才能讓我隨心所欲地寫著文章。

而第二個真正要感謝的一位關鍵人物，就是Cary，雖然他和我都是失敗的主管XDD，但真的很高興，人生之中，能被他帶到，因為有他，所以我才能在之前的公司好好的發揮，好好地摸索這些技術，也因為有他在前線當砲灰，所以才能讓我在後面無後顧之憂地亂搞，喔，不是，是專心的開發；當然，更重要的是，2010年3月的時候，第一次鼓起勇氣，抱著聽不懂的心情，想去微軟聽聽看軟體工程的課程( 軟體工程實務與軟體測試經驗談 )，抱持著問問看的心態問了老大Cary，而Cary只說了一句話，沒關係啊，就請公假做高鐵去聽聽看阿；於是這一聽，改變了整個後面的人生，謝謝你，Cary!。

其實，現在回憶起來，完全沒想到，那天的課程全都是大師級的，我就傻傻地抱持著聽聽看的心情，發抖地走進微軟；第一堂課的內容，我印象很深刻，想說怎麼有一個高高大大像流氓的人走進來 ( 胡百敬老師，對不起QQ，當初真的是這樣覺得的 )，但開始上課之後，就深深的感受到，百敬老師果然是活在不同領域的人，雖然當時他講的不是SQL Server XDD而是TFS，但真的讓我深深的入迷。

第二堂課，是彭靖灝老師，講的是軟體工程的北冥神功 ( 但內容我已經忘了XDD，老師對不起QQ )，上課的時候，一樣，就看著彭老師咻咻咻的，就把Demo一個接著一個完成，然後下課後，我還是恍神在那邊…

第三堂課，也是改變我人生對於軟體觀念的重要一堂課，是Ruddy老師的課，講的內容是TDD，這堂課，就像打通任督二脈一樣，笑話，課程內容，一直到現在，還在腦海裡揮之不去，為何有人可以不用投影片，為何有人可以亂跳著講XDD，為何有人能把課程變得那麼有趣，而直到今天為止，Ruddy老師的上課風格，也變成小弟我模仿的對象，也讓我感受到，Coding原來是件那麼快樂的事情，甚至，是可以做到老的事情，而那天，Ruddy老師也變成小弟的偶像XDD，我就這樣被Ruddy老師給捕獲了=v=。

其實，很難真正的寫出，這三堂課的重要性，只能說，真的很幸運的，這三堂課都是大師，就算一堆東西聽不懂XDD，或許真正收穫的東西不是那些技術，而是足以影響到我的人生的道理，直到今天，終於有機會，可以和這三位老師說，謝謝你們豐富的講課，開啟了小弟我程式道路的大門=v=。

後來，算是聽上癮了吧XDDD，常常動不動就盧著Cary，請著假去聽微軟的各個研討會、發表會，跑去台北上課、台中的VS2010發表會、高雄的VS2010發表會 ( 因為台中場聽到一半被招回，所以又跑去高雄場聽 )，而在高雄場聽Ruddy老師的課，也因舉手迅速XDD，Ruddy老師送了我一本測試之道這本書，爽了超久XDD。

後來，Cary大概看我聽上癮了，就幫我和公司申請微軟的TechDay 2010，這也是第一次參加這大型的聚會，而這一次，又改變了小弟的人生；在忙碌的會場中，第一次看到小朱，也聽了小朱前輩的Azure 安全課程，曹祖聖老師的SharePoint開阿，Ruddy老師的Azure，胡百敬老師的PowerPivot，肥馬的TFS，尤其是一開始的時候，肥馬在大會場展示的Demo時講話的聲音，到現在都還在腦海裡揮之不去阿XDD，也在這邊謝謝小朱前輩的指教，和肥馬不離不棄的陪著我XDD。

而在會場的現場，舉辦著MSDN抽獎，那是小弟我第一次看到大鈞哥，還有當初點部落的執行長Gipi老大，而也是這次，沒抽到MSDN XDD，所以下定決心，未來要好好的努力，當上MVP!!

晚上的IE 8 Party，更是有種無助的感覺XDD，小弟畢竟只有一個人，無依無靠的，一個人躲在角落，默默地啃著東西，眼光投射到MVP的桌子上，看著大家好開心，好快樂，也真的感受到，有一群愛好寫程式的同好們，真的好好，心裡想著，雖然不知道多久才有機會和大家在同一個桌上，但想認真地朝這個目標筆直前進，沒錯，我有了明確的目標；喔，對了，那次也是我第一次看到鼎鼎大名的Bill叔，Bill叔應該不知道XDD，當時我偷偷地躲在他的後面，看著他的背影XDD，也在這邊謝謝Bill叔。

而後，離場的時候，剛好又遇到以前打電動認識的朋友Random，看著他拿出MVP的名片，我又更想達成這個願望，也在此謝謝Random給我的建議。

隨著時間的推逝，小弟也開始慢慢地將一些東西利用Blog寫起來，也不斷地參加一些課程，每次，都是用仰慕的眼光，投射到台上的講師身上。

後來，Cary離開了舊公司，我也到了新公司，也利用一次的機會，認識了公司裡面唯一的MVP，Gipi老大 ( 其實當初是我一直纏著他XDD )，也因此，又展開了一段新的旅程，沒錯Gipi老大也是一個重要改變我的人，謝謝你，Gipi老大!

一年過後，又再次參加了TechDay 2011，這次有了之前的經驗，又加上認識了Gipi老大，於是就死纏著Gipi老大跑去聽Gipi老大的課程，那次，也是第一次看到91前輩，而當天晚上的第二屆SQL Hero，更是目睹了Rico拿下了冠軍=v= ( 也謝謝Rico和91後期的指導=v= )

事後的和MVP聊天的過程中，我跑去找了胡百敬老師和肥馬，頓時發現，胡百敬老師好友善喔，而且很喜歡和人聊天，從老師的眼中，看的出來，老師真的很愛很愛這些技術，因為只有很愛技術的人，才會滔滔不絕的，毫無保留的一直講，一直講，一直講；然後那天，也是肥馬人生悲劇的開始XDD，因為從那天開始，我就開始慢慢地纏上肥馬了。

而這次的TechDay 2011，讓我深深地感覺到，為什麼，台中都沒有類似的活動，台中為什麼都沒有這樣的社群，台中為什麼那麼可憐，所以就拖著Cary和Jim，來舉辦台中的社群；而這邊，也真的要感謝Jim，雖然你離微軟越來越遠XDD，但真的很感謝你一路陪我走來；還有James和Kande、Shawn，CCH、Laura無私的講課；DH和Janet無悔的支援。

辦的過程中，並沒有想像中的順利，講師找不到人講，沒甚麼人來願意聽，場地也借不到，但很感謝同事們，無怨無悔的付出，就算聽不懂，也願意出席當人頭，甚至幫忙找場地，也讓我沒有放棄，繼續地走下去。

而也要感謝Gipi老大，對於這個活動的鼓勵，並且邀請我去參加點部落的聚會，認識了大鈞哥、Jon哥，91哥，還有Chiachi，甚至介紹Kande和James來幫忙與支援講課，因為你們的幫忙，與推波，才有現在的我們。

最後，還要感謝非常關鍵的人物，那就是暗黑大大、Katherine，老實說，如果沒有暗黑前輩的推薦和Katherine一直塞報名表，老實說，我到現在還是沒有勇氣，投出申請書，也不會有今天的當選。

也很感謝一路上相陪與支持的舊同事和新同事們，雖然上面可能沒提到，但你們可能一句話就改變了我想法的前輩們( Lolota前輩、Bruce哥、Demo大、艾小克前輩、偉榮哥、Kevin哥、John哥、黃忠成老師、Jason前輩，德瑞克前輩，等等等。)，還有每位需要準備上課教材很久的講師們，還有每一篇文章都會深深影響小弟的神隱高手們，努力寫出好書的各位前輩們，非常感謝您們!!

( 如果真的有不小心遺漏，真的不是故意的QQ。)

最後的最後，小弟也知道，MVP其實不是個榮耀，而是一種責任，能力越強，責任越大，未來也會扛起這個責任，繼續的推廣，也希望讓更多人能親近這些技術，走著各位前輩走過的腳步，繼續努力，最後，再一次的，謝謝大家!!
