Windows Azure - 切換新版和舊版的管理介面

發表於 2012-07-24   |  

目前新版的管理介面是HTML5架構出來的，而舊版的管理介面是Silverlight架構出來的，當然，這東西也沒有甚麼好與壞，但是比較麻煩的是，目前新版的管理介面並非包含所有舊版的功能，而舊版的管理介面，也有很多新版的功能沒有；而最近整理Windows Azure文章的時候，發現常常需要切換來切換去，為了減少相關文章的篇幅，所以就把這篇文章獨立出來了… ( 嗯，小弟我是很懶的=v= )。
