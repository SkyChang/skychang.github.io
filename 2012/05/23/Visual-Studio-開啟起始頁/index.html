Visual Studio - 開啟起始頁

發表於 2012-05-23   |  

老實說，這篇真的沒什麼技術，但小弟我已經第三次遇到這個問題，然後每次都花半個小時在那邊找設定，所以一氣之下，決定把他記起來，至少未來遇到，不會再忘記了= =+

這篇的重點就如下圖，這個起始頁，這次又不小心被小弟關掉了，也就是說，開啟的時候，不會自動打開這個起始頁，雖然沒有起始頁還是可以正常的工作，但是，我就是想開工前偷看一下MSDN新聞阿!!

所以任性的小弟我，在這邊教大家( 我想大家搞不好也沒這需求XDD )如何恢復開啟Visual Studio的時候，預設開啟起始頁。

首先按下檢視，然後打開起始頁。

打開後，左下角就可以進行設定，把”啟動時顯示頁面”給勾起來，完工!!

總之，就這樣簡單，下台一鞠躬。
