Windows Azure - 申請3個月免費試用! ( 台灣篇 )

發表於 2012-06-03   |  

開始前的碎碎念，其實小弟之前有寫過一篇了，本來想用那篇文章做修改，但覺得那篇對小弟來說，太有紀念價值了，所以就決定複製那篇，並且編修後，變成了台灣篇這份=v=。

正文開始

Ruddy老師說過，使用Windows Azure，要拿老闆的卡，但是老闆死都不交出他的卡，所以幾番抉擇後，還是拿出自己的卡QQ…

申請Windows Azure的部分，小朱前輩這裡也有超級詳細的攻略說明，大家也可以參考小朱前輩的文章=w=。

備註:目前申請帳號需要用到手機驗證，而且只會傳到當地的手機號碼，如果沒有透過手機驗證，會無法繼續進行，而且亂Key電話號碼，會被鎖帳號!!另外，小弟因為已經申請過了，所以並沒有完全走過一次，如果中間有任何問題，可以留言給小弟，小弟在想辦法修改。

首先，當然是到微軟的Azure官方網站https://www.windowsazure.com/zh-tw/，目前的介面是滿漂亮的XDD，我們可以從”定價”的選單裡面看到”免費試用”，但是有一點要注意一下，目前Azure的免費試用申請，還是必須要有信用卡，而且需要當地的電話號碼!!

點下立即體驗後，會需要Windows Live的帳號!!。

接下來，會有三個步驟來處理，我們這邊可以看到，台灣已經出現了!!，而畫面上，也有再次的提醒大家，需要有電話號碼和信用卡。

這邊只要輸入電話號碼，就會傳簡訊過來，進行驗證；小弟之前因亂Key電話號碼，所以被鎖帳號，( 會變成，無論Key甚麼電話，都會告訴你電話錯誤。這是預防，有人惡意亂發簡訊 )，當你不小心被鎖定，就只能打電話求援了!!所以這部分要小心喔。

最後填入資料，填入完後，微軟會刷一塊錢( 不確定是一美元還是一塊台幣 )，來核對信用卡號，這要也要特別注意，未來如果更改資料，都會刷一塊錢來核對信用卡，小弟整整改了四次，改到信用卡公司都打電話過來給我，以為出了甚麼問題 = =。

通常到這邊就完成了，然後就可以體驗三個月的免費!!

後記

老實說，以前申請Windows Azure，吃了超多苦頭，而也一堆人，想用但是不知如何申請，這次台灣開始啟用服務後 ( 偷跑!? )，相信一定可以帶來一段Windows Azure的風潮!! 。
