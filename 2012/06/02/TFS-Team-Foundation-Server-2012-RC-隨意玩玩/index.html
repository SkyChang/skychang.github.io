TFS - Team Foundation Server 2012 RC 隨意玩玩

發表於 2012-06-02   |  

延續上一篇，畢竟現在還是RC，小弟就不做深入的介紹了，而且其實新的Web介面，小弟寫的Team Foundation Server雲端版的文章裡，也都有介紹了，等未來正式版出了以後，小弟會再重新整理一次文章，所以大家先稍微看看圖就好XDD。

前篇安裝完TFS 2012 RC，那我們這邊就稍微新增一個專案看看吧，首先，我們要先利用VS2012 RC連接到TFS 2012 RC吧!

連接的畫面也和以前一樣。

連接上後，VS 2012 RC的右邊，就會出現，小弟覺得這塊就弄得比VS 2010好很多。

因為還沒建立新的Team專案，所以我們先建立一個新的Team專案。

專案名稱就隨便取吧…=V=。

目前直接內建Scrum 2.0喔!! ( 1.0真的是難用到爆炸 )，目前是Preview 4，和Team Foundation Service ( Cloud版本 )是相同的。

這邊就繼續吧。

到這邊做個確定，如果確定，就按下完成繼續吧!!

然後就會開始建立。

到這邊就建立完成了。

我們就可以從VS 2012 RC看到Team Project已經建立起來了。

接下來，我們啟動Web管理介面來看看，如果是使用Windows Server，別忘了把IE的增強模式安全設定關閉掉，才能正常喔。

登入進去後，我們可以透過下面的全部瀏覽，來加入新建立好的Team Project。

這時候，我們會出現如下圖，我們就可以選擇剛剛建立好的Team Project。

進入後，就完全後Team Foundation Service ( Cloud ) 版本完全一樣了…

我們就可以直接在Web端新增Task之類的，詳細的介紹，大家可以去參考小弟的Team Foundation Service ( Cloud版本 )的文章，小弟就不多介紹了=v=。

後記
