日記 - Windows 8 RP、Windows Server 2012 、Visual Studio 2012、 TFS 2012 RC軍團 釋出了!

發表於 2012-06-01   |  

我想這幾天最大一件大事情，因該就是Windows 8 RP ( 因為沒加入2012軍團，所以才被稱為RP嗎 )、Windows Server 、Visual Studio、TFS 2012 RC軍團釋出了，想當然的，一定有很多前輩都裝好，而且也寫好開箱文了，所以小弟就不獻醜了XDD，純粹以日記的角度來寫這篇。

5/31號 上班中

就和平常一樣，窩在電腦前面工作，就在上班上到一半，突然看到了Lolota大大發表的訊息，原來是有MVP爆料，今晚就會出現Windows 8 RP了!( 這是一種故意的宣傳嗎= =? )，原本認為是6月初才會釋出的Win 8的小弟，完全沒有想到會提前釋出，然後馬上就開始猶豫，晚上要玩D3，還是要玩Win 8..( 其實比較想看看Visual Studio.. )

5/31號 pm11:00

因為在改Blog文章，所以一轉眼間就到了pm11:00，這時候，想說應該快要釋出了，準備搶頭香的小弟，另外開了一個瀏覽器，一邊用Google打上關鍵字，一邊用持續觀察Windows 8的官網，不知道為何，感覺比搶D3還刺激 ( 疑!? )

5/31號 pm11:3x

利用Google，搜尋到簡體的Windows 8已經悄悄地出現在MSDN了，也正式確定Visual Studio 11改為2012，既然發現到這個情報，小弟當然狂按F5刷新網頁，結果聽說沒多久，馬上就消失了…

6/1號 am00:00

還是沒出現，持續按F5…，這時候看到一線淑光，MVP官方丟出了一個訊息，訊息內容是說，在30min後，會有令人興奮的訊息!小弟看到後，當然持續等下去啦!

6/1號 30分鐘過後

的確是讓人興奮…MVP的官網更新了…Orz…這時又回到原點，只能繼續按F5…

6/1號 am02:00

受不了了，去睡覺去…

6/1號 am08:00

起床後，就發現釋出了…到底我等那麼久的意義是= =…

來看看Visual Studio 2012吧

喇了那麼多，其實小弟比較在意的是Visual Studio 2012，而且小弟相信，Win8的畫面，應該一堆人都會貼了吧QQ，所以小弟就不提了。

記得之前有提到，Visual Studio 11實在是太灰暗了，完整的表現出我們寫程式灰暗的人生，後來官方聽到抱怨太多，就加了一些色彩進去，現在讓我們就來看看Visual Studio 2012吧。( Windows 8 預設的桌布也不錯看喔=v=)

準備開始安裝嚕!!

安裝中。

這是完成後的畫面。

安裝完後，和以前一樣，需要設定環境，但為何這裡不用黑色哩= =。

到這邊，我們可以發現色彩變多了，至於版面配置，和之前的沒兩樣，至於功能面，就等其他前輩來補充了XDD。

在發布Visual Studio 2012之前，有提到新的樣板，小弟最期待的就是這個啦XDD，我們可以從下圖的位置去做變更。

Cool!，就變成像這樣的黑色了，感覺很像Expression Studio，更加的凸顯，我們寫程式的人，做到深夜，肝黑黑QQ；另外，因該有的Metro App也不會少。

但小弟最在意的是Azure的支援，Visual Studio 11並沒有對Azure支援，官方的網頁也說，如果要寫Azure，目前還是必須要用VS2010，所以這個版本特別期待，希望能包含進去；不過雖然有Cloud專案，但按下去後，還是要取得SDK，不過感覺有一點希望了喔。

按下去後，整個Orz…，又出現之前的那個網頁，好吧，畢竟現在也只是RC，就讓小弟期待6/7號的Azure大會，看看未來會不會出現SDK吧..

( 這裡也有人詢問。 )

我們可以看到，這次的Visual Studio 2012並沒有像之前的Visual Studio 11包山包海了。

後記

其實這一天，也算不錯的體驗啦；但是這次Visual Studio 2012暫時不支援Azure還是一件很可惜的事情，但相信未來的不久，就會有完整的支援了。
