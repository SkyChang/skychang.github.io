紀錄文 - Windows 8和Visual Studio 2012 正式釋出

16 August 2012 — Written by Sky Chang
#Sky的IT碎碎念#TFS#Visual Studio#Windows

Windows 8 和Visual Studio 2012 RTM終於美西時間2012/8/15於MSDN釋出 ( 台灣為8/16 )，當然，這篇一樣不會講到甚麼新功能之類的，因為有太多前輩和專門的作家們會寫XDD，所以小弟就不在這邊介紹Windows 8和Visual Studio 2012；那既然如此，為何要寫這篇呢?，其實只是單存的希望，能記錄一下這次的過程，因為這次的意義，對小弟來說也是非同小可，所以想抱著寫日記的心情，在這邊紀錄一下  ( 偶而讓小弟寫個紀錄文吧XDD )。

自從去年TechDays 2011的時候，剛好也是Windows 8的Preview的發表，其實當初也沒想到會一路這樣裝下來，還記得那時候，利用Windows 8 Preview招搖撞騙XDD，當作講課回答問題的禮物；當然，自己當時也利用VM，小玩了一下，當初只單純的覺得，真難用。

後來，隨著Beta，RP，一路走來，也看到了Visual Studio，從灰暗的人生，變成黑暗的人生XDD ( 但老實說，是真的很喜歡黑色風格 )，也因為當初Visual Studio 2012 Beta不支援Azure開發，所以也沒拿來當正式環境使用。

到了Windows 8 RP和Visual Studio 2012 RC，就莫名其妙地變成正式的環境了 ( 反正我的需求也只需要上上網，寫寫Code… )，也從原本覺得很難用的想法，慢慢理解為什麼Win 8長這樣，也很期待平板的到來，甚至，換回Win 7的時候，感覺開始的內容變得很雜…變得很難用… ( 老實說，用到Win 8，幾乎把常用的東西都釘到工作列上了，幾乎很少進去裡面找程式 )，總而言之，就是習慣了啦XDD，所以，人真的很容易習慣的動物…

而這段時間，其實感觸最大的是，以前重灌系統的時候，是多麼的麻煩，一堆東西要備份，一堆設定要調教，反正整個系統弄到順暢，大概需要一個月的時間，而如今，Win 8支援Windows Live後，一些設定我也不用調了，甚至現在有了SkyDrive和Dropbox，再加上Team Foundation Service來存放Source Code，重灌根本也不用管啥備份… ( 好啦，可能我需求比較低… )，所以一路走來，也算順順暢暢，只能說，時代真的進步了。

而這個時間點，小弟我能下載這個版本，其實對我也是意義重大 ( 所以才會有這篇 )，因為從來沒想到，我也可以更上這個時代的腳步XDDD。

最後，或許四年後，Win 10 ( 因該不會取那麼難聽的名子 )也出了，也不知道未來的世界會變成怎樣，但站在科技今天的這個日子裡，期待著年底的平板，雖然也不知道Win 8這個雙面刃最後到底是好是壞，但只想說，今天能活在這個時代，真好!!
