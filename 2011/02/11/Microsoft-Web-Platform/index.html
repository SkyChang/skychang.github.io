Microsoft Web Platform

11 February 2011 — Written by Sky Chang
#ASP.NET MVC#Tools#Visual Studio

想要裝個Silverlight 4 Tools卻不知道這個版本是不是最新的!? ASP.NET MVC 3 出了，還在找安裝程式。

好吧，我承認我以前常常幹這種事情，找東找西，就只是為了找一個新版的套件，找的了，還懷疑東懷疑西，繼續尋找看看，是不是最新版。

但這些問題，碰到Microsoft Web Platform就都解決了!!

官方的解釋是這樣 官網位置http://www.microsoft.com/web/

Microsoft Web Platform 提供的不只是功能強大的一組工具、伺服器和技術，更提供了可打造主機網站、服務和應用程式的完整生態系統。 無論您是想要打造公司的網站呈現，或是要啟用新一代的 MySpace，我們都可提供讓您美夢成真的產品。 榮獲財星五百大公司信任，擁有數以百萬的個人使用者的 Microsoft Web Platform，就是完美工作的好夥伴﹗

我們直接先來看看Web Platform執行的畫面吧! ( 目前Web Platform Install 3.0會找不到一些比較新的工具或是測試中的工具，例如:SQL Server Data Tools，目前只有Web Platform Install 4.0 Beta才有提供，有需要的人可能要注意一下。 ) 如畫面所示，應該可以看到像是.NET 4、MVC3等等，都可以輕易地在Web Platform選單上面看到了! 而我們只要在Web Platform上輕鬆地按下新增，並且安裝，Web Platform就會自動地幫我們處理好。 簡單吧!，超適合像我這種懶人使用=w=。

那如何下載安裝呢? 我們只要到Web Platform之官網http://www.microsoft.com/web/ 並且點選下載。 然後會進到下載頁，點下載，並且下載完後安裝即可。 裝完後執行後，就如同第一個畫面一樣，就可以開始在Web Platform上簡單的將您想要的東西裝上去啦!! ( P.S 若要移除安裝上去的套件，目前在Web Platform 3.0上是不行的， 還是只能透過程式管理員移除. )

不知道大家有沒有這種經驗， 平常Server沒有安裝ASP.NET MVC 3的話，就必須在發佈的時候， 上傳一堆ASP.NET MVC 3之DLL檔到網站bin目錄， 要不然就是要去微軟的官網找ASP.NET MVC3安裝程式， 但一不小心，也可能裝到的是RC版本， 現在有了Web Platform只要輕輕地按下新增和安裝，就可以非常方便的安裝， 小弟我不但開發機有裝Web Platform，Server上我也裝了Web Platform。 所以現在我們實際來利用Web Platform裝一下最夯的ASP.NET MVC 3吧。

如果已經裝了Visual Studio 2010 SP1，則內建就會有MVC 3，如果沒有裝SP1的人，可以考慮直接使用Web Platform直接裝SP1

首先，我們先選擇產品 -> ASP.NET MVC 3 新增 -> 安裝。

然後他會和你說，要裝哪些東西

確定後，Web Platform就會開自己跑了。

過一會兒，就安裝完成。

Web Platform就是如此的簡單好用!
