Windows Server的磁碟清理工具

發表於 2011-11-07   |  

標題下的感覺很聳動，但其實這是我今天發生很瞎的一件事情…，磁碟清理工具可能對現代的人來說幾乎很少使用了，因為硬碟容量動不動就幾T，但對於SSD的使用者，或是空間不足的人，就會很敏感，也常常會執行磁碟清理工具，來將暫存的檔案刪除，下圖就是磁碟清理工具。

而今天，很瞎的，我在Windows Server 2008 R2 ( 2008版本應該也一樣 )，找不到磁碟清理工具!!，那時候我還在想，是不是Server版本的位置不一樣…，後來才查到，原來是要安裝桌面體驗，才會出現此工具= =…

安裝完後，就可以正常的使用磁碟清理工具了…
