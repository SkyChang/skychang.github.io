Windows Azure 新增空白Web Role的作法

發表於 2011-10-31   |  

每次新增Windows Azure專案的時候，可以在下圖選擇Web Role或是Work Role等等之類的，但是我其實不是很喜歡在這邊選擇。因為這裡加上去的專案，都會帶太多範例進來，例如選擇了ASP.NET Web角色 ( 也就是Web Role )的話，預設會將login、首頁等等的範例進來，每次都刪刪減減，所以這邊提供一個增加空白Web Role的作法。

這裡就不選擇了，讓右邊視窗框空白吧。

[](http://lh5.ggpht.com/-59c6jygQBCY/Tq9Vs-QCcmI/AAAAAAAAAzk/7SznEJa9PAI/s1600-h/image_thumb%25255B8%25255D%25255B2%25255D.png)

如果選擇了ASP.NET Web Role，會帶進來太多的範例，例如App_Data、Account的程式、About.aspx、還有長長串的Web.Config。

[](http://lh6.ggpht.com/-ol_QSSEF5Hs/Tq9VuwHn3dI/AAAAAAAAAzw/mV9mWE_aRn8/s1600-h/image_thumb%25255B9%25255D%25255B2%25255D.png)

所以我會先不選，最後再自己加專案進來，下圖是完全沒選的狀況，空空是也!

[](http://lh5.ggpht.com/-AnCR-40HxaM/Tq9VwOThawI/AAAAAAAAA0A/5ry4wUK28sY/s1600-h/image_thumb%25255B10%25255D%25255B2%25255D.png)

然後可以再新增專案。

[](http://lh3.ggpht.com/-mWiJ9QBlqHg/Tq9VxaI6pRI/AAAAAAAAA0Q/IN__HrKjcT4/s1600-h/image_thumb%25255B13%25255D%25255B2%25255D.png)

然後就可以選擇空白的ASP.NET Web 專案。

然後在原本的Windows Azure專案，再去選擇剛剛增加的專案。

然後按下確定就可以了。

這樣，就會將Web Role弄進來。

但還要注意一些小地方，Web Role是我們自己用加進來的，所以會有些東西沒有參考到，這部分要請大家以下這三個Include進來。

  • Microsoft.WindowsAzure.Diagnostics
  • Mictosoft.WindowsAzure.ServiceRuntime
  • Mictosoft.WindowsAzure.StorageClient

這樣就完成了!!
